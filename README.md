# 📝 Менеджер Задач (FastAPI ToDo App)

## 🚀 Основной функционал

* **Регистрация и вход:** Каждый пользователь создаёт свой аккаунт. Пароли не хранятся в открытом виде — шифруются с помощью библиотеки `bcrypt`.
* **Личный список задач:** Каждый видит только свои задачи. Чужие задачи посмотреть нельзя.
* **Управление задачами:** Задачи можно добавлять (с описанием, категорией и дедлайном), отмечать галочкой как выполненные и удалять.
* **Редактирование названий:** Если в названии задачи допущена ошибка, её можно переименовать прямо на ходу.
* **Умное приветствие:** Сайт смотрит на время суток и пишет «Доброе утро», «Добрый день», «Добрый вечер» или «Доброй ночи».

---

## 🛠 Стек технологий

* **Бэкенд:** Python 3, FastAPI, Uvicorn (ASGI-сервер)
* **База данных:** SQLite, SQLAlchemy ORM (для управления таблицами пользователей и задач)
* **Фронтенд:** HTML5, CSS3 (Кастомные стили в темной неоновой теме), JavaScript (Vanilla JS), шаблонизатор Jinja2

## 💻 Инструкция по локальному запуску

Для того чтобы запустить проект на своем компьютере, выполните следующие шаги:

### 1. Клонирование репозитория
Склонируйте проект к себе на ПК:
```bash
git clone https://github.com
cd ИМЯ_РЕПОЗИТОРИЯ
```

### 2. Настройка виртуального окружения
Создайте изолированное виртуальное окружение и активируйте его:
* **Для Windows (PowerShell / CMD):**
  ```bash
  python -m venv .venv
  .venv\Scripts\Activate.ps1
  ```

### 3. Установка зависимостей
Установите все необходимые библиотеки одной командой (они автоматически подтянутся из файла `requirements.txt`):
```bash
pip install -r fastapi_todo_app/requirements.txt
```

### 4. Запуск приложения
Запустите сервер автоматической перезагрузки Uvicorn:
```bash
uvicorn fastapi_todo_app.app:app --reload --port 8080
```
После запуска откройте в браузере страницу: [http://127.0.0.1:8080](http://127.0.0.1:8080)
